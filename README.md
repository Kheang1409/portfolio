# frontend

Clean, minimal Next.js + TypeScript SPA port of the existing Angular frontend.

Quick start

1. cd frontend
2. npm install
3. npm run dev

Build for production

1. npm run build
2. npm start

Notes

- This is a lean scaffold focused on clean, DRY code and a simple structure.
- Add assets to `public/` and global styles in `src/styles/globals.css`.
